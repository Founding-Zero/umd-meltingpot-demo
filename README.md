# meltpot_demo

## Installation

```
make install
```

## Usage

to start:

```
python meltpot_demo/harvest/play_harvest.py --observation WORLD.RGB
```

to run experiments:

```
meltpot_demo --example-config '{"block_size": 10}'
```

```
meltpot_demo --config $PWD/runs/<YYYY-MM-DD>---<HH-MM-SS>/config.json
```
